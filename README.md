# nextjs

_nextjs_ é apenas uma demonstração de um projeto Next.js/React, utilizando Atomic Design para organização dos componentes, axios para as chamadas de API, Styled Components para estilização e RTL nos testes.

- https://nextjs.org/docs
- https://pt-br.reactjs.org/docs/getting-started.html
- https://bradfrost.com/blog/post/atomic-web-design/
- https://github.com/axios/axios
- https://styled-components.com/
- https://testing-library.com/docs/react-testing-library/intro
